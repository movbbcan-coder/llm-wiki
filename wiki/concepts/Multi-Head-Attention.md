# Multi-Head Attention

## Overview

**Multi-Head Attention** runs multiple [[Attention Mechanism]] operations in parallel with different learned linear transformations.

## Purpose

Allows the model to:
- Attend to information from different representation subspaces
- Capture different types of relationships simultaneously
- Improve model expressiveness

## Usage

Core component of [[Transformer]] architecture, working alongside [[Self-Attention]].
