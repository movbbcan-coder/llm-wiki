# Self-Attention

## Overview

**Self-Attention** is a mechanism where an input sequence attends to itself, allowing each position to consider all other positions when computing its representation.

## How It Works

Uses the [[Attention Mechanism]] with Query-Key-Value system, but all three come from the same input sequence.

## Significance

Core component of the [[Transformer]] architecture, enabling:
- Parallel processing of sequences
- Capturing long-range dependencies
- Context-aware representations
